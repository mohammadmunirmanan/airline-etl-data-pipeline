# Airline Ticket Sales ETL Pipeline

## Project Overview
This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline for airline booking and sales data using Python, SQL, and Power BI.

The project focuses on transforming raw airline booking data into meaningful business insights through data cleaning, transformation, storage, and visualization.

---

## Technologies Used
- Python
- SQL
- Pandas
- Power BI
- Git/GitHub

---

## Project Workflow

### Extract
Loaded airline booking data from CSV files.

### Transform
Cleaned and transformed datasets using Python and Pandas:
- Removed duplicates
- Handled missing values
- Standardized formats
- Created calculated fields

### Load
Loaded transformed data into SQL database for reporting and analysis.

### Visualization
Built Power BI dashboards to analyze:
- Ticket sales
- Revenue trends
- Customer activity
- Flight performance
